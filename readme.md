# framework()

framework() is a personality-based productivity system built to explore and practice frontend web development using HTML, CSS, and JavaScript.

This project was built to improve my understanding of the following in 2024:

- Modular JavaScript structure
- UI layout systems
- Conditional rendering
- Component-based thinking (without frameworks)
- Theme-based styling
- Basic analytics visualization

---

## Project Idea

The idea behind framework() is simple:

Different people think and work differently.

Instead of using one fixed productivity layout, this project simulates five different “brain types”:

- Architect
- Sprinter
- Dreamer
- Juggler
- Minimalist

Each brain type has its own dashboard layout and focus system, while sharing a common task engine underneath.

---

## Tech Stack

- HTML  
- CSS (base + component + theme structure)  
- Vanilla JavaScript  
- localStorage (for session simulation)

No frameworks were used — the goal was to understand how everything works at a fundamental level.

---

## What I Learned

Through this project, I practiced:

- Structuring a multi-page frontend project
- Organizing code into logical modules
- Separating logic from presentation
- Building reusable components
- Creating different layouts from the same data
- Implementing simple analytics visualizations
